Plugin will list alphabets A-Z on hover over alphabets it will list Post/Page/Category names.


== Description ==

1.plugin will list alphabets A-Z.

2.When user hovers over any alphabet it will lists posts, pages title starting with
the alphabet user hovers.

3.The titles will have link and it will take user to single page.

you need to use a shortcode to embed the listing in a post or page.

Example:
1. [atoz]
2. [atoz type="page"]
3. [atoz type="category" title="A-Z Category"]
note: default values are
type="post"
title="A to Z listing"

This plugin is brought to you by [WPoets](http://wpoets.com/ "WordPress based development shop") development team.

== Installation ==

1. Upload 'listing by alphabet' folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `**[atoz]**` shortcode in the wordpress editor.

== Remove plugin ==

1. Deactivate plugin through the 'Plugins' menu in WordPress
2. Delete plugin through the 'Plugins' menu in WordPress

It's best to use the build in delete function of wordpress. That way all the stored data will be removed and no orphaned data will stay.
